2.2.16 (2020-08-06)
-------------------

* Improve instability of dynamic reconfigurable options.

* Contributors: doronhi

2.2.15 (2020-07-13)
-------------------
* Check runtime version of librealsense2 vs. compiled version and issue a warning is mismatch occurs.

* add d435i-xacro
